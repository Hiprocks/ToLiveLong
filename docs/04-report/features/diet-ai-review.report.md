## 3. 핵심 설계 결정

### 왜 모달 전환을 선택했는가
- 인라인 평가가 차트 탐색을 방해해 통계 페이지 본래 목적과 충돌.
- 모달 전환으로 차트 탐색과 AI 해석을 분리해 사용 흐름을 단순화.

### 왜 로컬+서버 이중 저장을 선택했는가
- 로컬만 사용하면 디바이스 변경 시 평가가 사라지는 문제가 발생.
- 서버 백업을 추가해 디바이스 간 복원을 보장하면서, 로컬 캐시로 즉시 표시 성능을 유지.

## 4. 사용자 가치

- 차트 사용성 저하 없이 필요 시 즉시 AI 해석 확인 가능.
- 재생성 전까지 동일 평가를 유지해 비교/회고가 쉬워짐.
- 기기 변경/브라우저 변경 시에도 마지막 평가 복원 가능.
