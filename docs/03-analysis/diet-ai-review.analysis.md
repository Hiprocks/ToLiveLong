## 2. 구현 품질 분석

### UX/흐름
- 통계 차트 흐름을 깨던 인라인 텍스트를 제거해 핵심 화면 밀도를 회복함.
- 액션을 `받기/보기/재생성`으로 명확히 분리해 상태 이해 비용을 줄임.
- 모달 위치를 중앙 고정해 디바이스별 위치 편차를 제거함.

### 데이터 일관성
- 로컬(`localStorage`) + 서버(`user` 시트 JSON 스냅샷) 이중 저장 적용.
- 초기 로드에서 서버 스냅샷을 읽고 로컬 캐시와 비교해 최신값 우선 적용.
- 재생성 완료 시 서버 저장 API를 동기 호출해 디바이스 변경 시 복원 가능.

### 안정성
- 마크다운 잔여 기호 노출 이슈를 프론트 sanitize + 프롬프트 제약으로 이중 차단.
- `build`/TS/기존 테스트(13/13) 모두 통과로 회귀 없음 확인.

## 3. 수용된 변경사항
- `AI평가 보기` 버튼 하단에 평가 범위 라벨 `(M/d~M/d)` 표시.
- `user` 시트 직렬화에 `dietReview` JSON 컬럼(확장 필드) 추가.
- `GET/PUT /api/sheets/user/diet-review` 신규 경로 추가.
